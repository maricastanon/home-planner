📖 What Is This File?

The complete X-ray of your app. Every tab, button, function, variable, style, connection, and bug — visible, labeled, and explained in plain English. Designed so you can understand, modify, and port features yourself.

🧬 The Three Layers

🌸 HTML — Skeleton

The visible structure. Buttons, containers, inputs. What the user sees and touches.

🌿 JS — Brain

The logic. Functions that run on click, type, or load. What makes things happen.

🎀 CSS — Skin

Colors, sizes, spacing. How things look. Uses var(--name) for reusable values.

📐 HTML Nesting

<div id="app"> ← outermost box ├── <nav id="tab-bar"> ← navigation │ ├── <button>Tab 1</button> │ └── <button>Tab 2</button> ├── <div id="tab-1"> ← tab content │ ├── <div class="toolbar"> │ │ └── <button>Action</button> │ └── <div class="content"> └── <div id="modal"> ← popup

🔗 Golden Rule

Rename an id in HTML → also rename in JS & CSS. Rename a function → update onclick. Delete a CSS variable → elements using it break.
This anatomy maps all three layers for every feature.
